# 🚀 Intellia AI

An AI-powered learning platform that provides intelligent conversations, interactive diagrams, and quizzes to make learning more engaging.

## 🌐 Live Demo

🔗 https://intellia-ai.vercel.app/auth/signin

## ✨ Features

- 🤖 AI Chat Assistant
- 📊 Diagram Visualizer
- 📝 AI Quizzes
- 💬 Follow-up Questions
- 🔐 Secure Authentication
- 📱 Responsive Design

## 🛠️ Tech Stack

- Next.js
- React
- Tailwind CSS
- Supabase
- Google Gemini AI
- Groq API

## 🚀 Run Locally

```bash
git clone https://github.com/vaishnavi121001/intellia-ai.git
cd intellia-ai
npm install
npm run dev
```

Open: `http://localhost:3000`
